## Correr Local

Tener istalado [Heroku Toolbelt](https://toolbelt.heroku.com/).

```sh
$ git clone https://github.com/leonfloch/Programa4.git
$ cd Programa4
$ mvn install
$ heroku local web
```

Para correr la aplicación: [localhost:5000](http://localhost:5000/)
